# 💰 Sistema Bancário em Python

Projeto desenvolvido como desafio prático da plataforma **Digital Innovation One (DIO)**, com o objetivo de aplicar conceitos fundamentais de Python e lógica de programação.

---

## 🚀 Funcionalidades

O sistema simula operações básicas de um banco, permitindo:

* 💵 Depósito de valores
* 💸 Saque com limite diário
* 📄 Visualização de extrato
* 👤 Cadastro de usuários
* 🏦 Criação de contas bancárias
* 📋 Listagem de contas cadastradas

---

## 🧠 Conceitos aplicados

Durante o desenvolvimento, foram utilizados:

* Funções em Python
* Estruturas condicionais (`if`, `elif`, `else`)
* Laços de repetição (`while`)
* Listas e dicionários
* Manipulação de strings
* Boas práticas de organização de código

---

## ⚙️ Regras de negócio

* Limite de saque: **R$ 500 por operação**
* Máximo de **3 saques por dia**
* Apenas valores positivos são aceitos
* Cada usuário é identificado por um **CPF único**
